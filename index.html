What version does Eclipse Network server support?
Eclipse Network supports Minecraft version: 1.20, you may also use any other older or lower versions to connect and play on the server.
Where is Eclipse Network minecraft server hosting located?
Eclipse Network is currently hosted in Singapore and has a great connection.
How do I play on the Eclipse Network Minecraft server?
Open the launcher, next click the "Play" button then select "Multiplayer" from the menu.

Click "Add Server" tab and input the text play.dreamers.fun on the "Server Address" box then hit "Done".

Once the connection is available and turns green you may click on "Join Server" button to play on Dreamerz Network.
What is the server IP for Eclipse Network?
